# Datos Masivos

La BD con la que se trabajará es [esta](https://github.com/GemaGuerraV/DatosMasivos/blob/main/yellow_tripdata_2025-01.parquet).

- [Tarea #1](https://github.com/GemaGuerraV/DatosMasivos/blob/main/notebooks/Tarea1_DM.ipynb)

  Se usa PySpark para filtrar datos, generar estadísticas descriptivas básicas y realizar algunas operaciones aritméticas entre registros y columnas.

- [Tarea #2](https://github.com/GemaGuerraV/DatosMasivos/blob/main/notebooks/Tarea2_DM.ipynb)

  Se preprocesa la base de datos, se convierte el origen de datos a RDD con pySpark y se realiza operaciones en el RDD, como estadísticas descriptivas básicas.

- [Tarea #3](https://github.com/GemaGuerraV/DatosMasivos/blob/main/notebooks/Tarea3_DM.ipynb)

  Se realiza manipulación de filas y columnas en los datos elegidos, se modifican datos se agregan nuevas columnas calculadas se filtran resultados.

- [Tarea #4](https://github.com/GemaGuerraV/DatosMasivos/blob/main/notebooks/Tarea4_5_DM.ipynb)

  Se realiza análisis con MLlib de pyspark al conjunto de datos.
